# Introduction to Generating PDFs
 - PDF reports help control output formatting and presentation
 - Python provides tools to generate PDFs automatically
 - ReportLab is a library for creating PDF documents
 - PLATYPUS offers high-level PDF layout and typography tools
 - Data like dictionaries can generate structured PDF reports

from reportlab.platypus import SimpleDocTemplate
report = SimpleDocTemplate("/tmp/report.pdf")

from reportlab.platypus import Paragraph, Spacer, Table, Image

from reportlab.lib.styles import getSampleStyleSheet
styles = getSampleStyleSheet()

report_title = Paragraph("A Complete Inventory of My Fruit", styles["h1"])

report.build([report_title])
